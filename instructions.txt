Steps to launch the bot:
Docker must be installed on the host
There should be files in one directory: bot.py config.ini Dockerfile Logger.Log Logger.py parsers.py require.txt
1. docker build -t bot.py.
2. docker run -d -v $(pwd)/Logger.Log:/app/logger.log -v $(pwd)/config.ini:/app/config.ini bot.py (in case of Linux)
docker run -d -v ${PWD}/Logger.Log:/app/logger.log -v ${PWD}/config.ini:/app/config.ini bot.py (in case of Windows)
